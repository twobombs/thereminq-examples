# Prototyping

This directory contains prototyping scripts for Out-of-Time-Order Correlators (OTOCs) and Quantum Echo experiments using PyQrack.

## Files

*   `otac-claude.py`: Script potentially related to OTOC calculation (possibly generated or inspired by Claude).
*   `otoc_validation_isingonly.py`: Validation script for OTOC in Ising models.
*   `otoc_validation_isingonly.sh`: Shell script to run the OTOC validation for Ising models.
*   `otoc_validation_isingonly_graph_highest.py`: Variant of the Ising OTOC validation script.
*   `otoc_validation_isingonly_graph_mpio.py`: Variant of the Ising OTOC validation using MPIO (Matrix Product Isomeric Operators?).
*   `otoc_validation_isingonly_graph_mpio_zero.py`: Variant of the MPIO validation script.
*   `otoc_validation_isingonly_graph_mpio_zero_shape.py`: Another variant of the MPIO validation script.
*   `otoc_validation_isingonly_graph_mpio_zero_shape_echo.py`: MPIO validation script with echo.
*   `otocs-prediction.py`: Script for predicting OTOC values.
*   `otocs-pyqrack.py`: PyQrack implementation for OTOC calculation.
*   `otocs-pyqrack2d.py`: 2D variant of the PyQrack OTOC implementation.
*   `quecho-pyqrack.py`: Quantum Echo implementation using PyQrack.
