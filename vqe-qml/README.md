# Vqe-qml

This directory merges Variational Quantum Algorithms with Quantum Machine Learning applied to generalized Hermitian matrices (`hermitian-matrices-qiskit.py`, etc.). It employs parameterized quantum circuits to learn the eigen-spectrum of abstract, non-chemical Hamiltonians. These scripts test gradient descent optimizers on purely mathematical objective functions, serving as primitives for quantum principle component analysis.
