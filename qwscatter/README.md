# Qwscatter

This directory simulates and visualizes 1D and 2D Quantum Walks. Scripts like `qwscatter-large-qrack-multi.py` leverage PyQrack to parallelize the evolution of large grid dimensions. Unlike classical random walks, quantum walks exploit phase interference to achieve ballistic spreading (super-diffusive behavior), serving as foundational primitives for spatial search algorithms and network routing protocols.
