# Quantinuum Benchmarks

This directory contains a collection of benchmarks and experiments related to the Transverse Field Ising Model (TFIM), likely run on or for Quantinuum hardware.

## Subdirectories

*   **`island-run/`**: Contains a collection of benchmarks and experiments related to the "islands" of qubits on a quantum device.
*   **`run352/`**: Contains a snapshot of a specific benchmark run, labeled "352".
*   **`scans/`**: Contains a collection of benchmark runs with different SDRP (Stochastic Drift-Diffusion Process) parameter settings.
