# Benchmarks

This directory contains a collection of benchmarks for various quantum computing algorithms and models.

## Subdirectories

*   **`ising/`**: A collection of benchmarks and experiments related to the Ising model, a fundamental model in statistical mechanics used to study ferromagnetism.
*   **`rcs/`**: Benchmarks related to Random Circuit Sampling (RCS), a task that is believed to be computationally difficult for classical computers.
