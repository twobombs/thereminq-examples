# Positron

This directory implements `unitary-inverse-metrology-example.py`, a script that explores quantum time reversal and inverse unitary operations. Inspired by relativistic analogies (e.g., "anti-qubits" or positrons), the codebase demonstrates how applying the complex conjugate transpose of an evolution operator ($U^\dagger$) can rewind quantum dynamics. This is fundamentally used in quantum metrology, dynamical decoupling, and protocols like Loschmidt echoes to reverse coherent errors.
