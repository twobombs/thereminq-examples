# Python Ising Simulations

This directory contains Python-based simulations and visualizations of the Ising model.

## Subdirectories

*   **`ising_ace_islands-check/`**: Contains a collection of scripts for running and visualizing ACE Ising depth tests, with a focus on checking the "islands" of qubits.

## Visualizations

This directory contains a variety of visualizations of the Ising model simulations, including 2D and 3D heatmaps, line plots, and other graphs.
