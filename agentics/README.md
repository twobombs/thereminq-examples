
# Agentics

This directory contains scripts designed for use in ThereminQ-Tensor containers.

**WARNING: Do not run agentic code on your local machine with full permissions.**

## Scripts

*   `gemini-query.py`: A Python script to query the Gemini API, likely for testing or agentic interaction.
*   `simple-gemini-query.py`: A simplified version of the Gemini query script.
