# Weed

This directory explores holographic embedding algorithms in tensor networks (`generate_sparse_density_data.py`, `feed_weed.py`). Operating under paradigms derived from AdS/CFT correspondence, these scripts simulate the ingestion and bulk-boundary matching of quantum states. It compresses the bulk degrees of freedom into boundary topological representations, validating techniques to embed complex intra-patch entanglement into manageable boundary baths.
