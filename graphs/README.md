# Graphs

This directory contains a suite of automation and visualization scripts (e.g., `metriq2025estimate.py`, `sycamore_rings.sh`) designed to process telemetry, profile performance, and generate topological representations of quantum circuits. These tools map out logical qubit interactions, compute entanglement graphs, and render 3D visualizations (like QFT circuits or Sycamore patch topologies). This facilitates hardware-aware circuit analysis, optimizing routing, crosstalk mitigation, and mapping algorithms to specific native qubit topologies.
